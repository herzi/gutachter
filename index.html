gutachter by herzi

A graphical user interface to glib/gtk test cases…

Gutachter will provide you with a very easy and straight-forward way of running your unit tests…
Start Gutachter, open your test suite and let it run the tests every time the test suite gets updated…

Authors

  • Sven Herzberg (all of the code, recoloring of the icons)
  • Andreas Nilsson (original icons)

Installation

  • Fedora 12
    • Install the repository specification.
    • Install the gutachter package.
    • This will install the gutachter package and will also provide you with updates via yum/package-kit.
  • For other distributions supported by the OpenSUSE build service, please open a bug and request a package.

You can download different versions of this project:

  • the latest development version (BEWARE: this might be unstable)
  • Version 0.0.1

Contribute
