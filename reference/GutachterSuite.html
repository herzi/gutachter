enum GutachterSuiteStatus

typedef enum
{
  GUTACHTER_SUITE_INDETERMINED,
  GUTACHTER_SUITE_LOADING,
  GUTACHTER_SUITE_LOADED,
  GUTACHTER_SUITE_RUNNING,
  GUTACHTER_SUITE_FINISHED,
  GUTACHTER_SUITE_ERROR
} GutachterSuiteStatus;

Represents the internal state of the GutachterSuite.

GUTACHTER_SUITE_INDETERMINED

 the suite is freshly created

GUTACHTER_SUITE_LOADING

 the suite is currently loading the list of tests

GUTACHTER_SUITE_LOADED

 the suite is done loading

GUTACHTER_SUITE_RUNNING

 the suite is being executed right now

GUTACHTER_SUITE_FINISHED

 the suite has been executed and is finished

GUTACHTER_SUITE_ERROR
