Widget Lookup — functions for looking up widgets

Description

These functions are useful to lookup widgets in tests. Imagine this widget structure: 

1
2
3
4
5
 
GtkWindow (title "main window")
 + GtkHBox
    + GtkLabel
    + GtkEntry
    + GtkButton

You can do these kinds of lookups: 

  • 1
    		 
    gutachter_lookup_child(NULL, "")

    will return NULL 

  • 1
    		 
    gutachter_lookup_child(NULL, "urn:gtk:")

    will return NULL 

  • 1
    		 
    gutachter_lookup_child(window, "")

    will return the GtkWindow 

  • 1
    		 
    gutachter_lookup_child(window, "urn:gtk:")

    will return the GtkWindow 

  • 1
    		 
    gutachter_lookup_child(NULL,"urn:gtk:GtkWindow(\"main window\")")

    will return the GtkWindow 

  • 1
    		 
    gutachter_lookup_child(hbox, "")

    will return the GtkHBox 

  • 1
    		 
    gutachter_lookup_child(hbox, "urn:gtk:")

    will return the GtkHBox 

  • 1
    		 
    gutachter_lookup_child(window, "urn:gtk:GtkWidget[0]")

    will return the GtkHBox 

  • 1
    		 
    gutachter_lookup_child(label, "")

    will return the GtkLabel 

  • 1
    		 
    gutachter_lookup_child(NULL,"urn:gtk:GtkWindow(\"main window\"):GtkWidget[0]:GtkWidget[2]")

    will return the GtkButton

gutachter_lookup_child ()

GtkWidget*          gutachter_lookup_child              (GtkWidget *widget,
                                                         gchar const*path);

Lookup a child of widget (or from the toplevel if widget is NULL).

widget :

 the parent GtkWidget or NULL

path :

 the path to the desired widget

Returns :

 the child specified by widget and path or NULL if not found.

gutachter_lookup_widget ()

GtkWidget*          gutachter_lookup_widget             (gchar const*path);

Look up a widget from the top level. This is the same as gutachter_lookup_child() with NULL as the first parameter.

path :

 the path (from the toplevel) to the child

Returns :

 a GtkWidget if a child was found, NULL otherwise.
